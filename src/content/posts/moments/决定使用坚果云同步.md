还是想要实现手机端写动态，刷到一篇[知乎文章](https://zhuanlan.zhihu.com/p/1983944140099167328)，决定放弃 git 和 remote sync，使用 nutstore sync。

git 在手机上用比较麻烦，用 obsidian 的 git 插件我好像 push 不上去，用另外的 gitsync 又遇到网络问题。

remote sync 是我之前玩 hacdias webdav 弄的，需要保持电脑运行，切不在同一局域网要 tailscale 穿透，比较麻烦。

转换思维，其实我不需要一定直接用 git 来作同步，git 只负责推送到 github 部署，坚果云只负责同步。

其实我之前有听说过用坚果云的，但是听说有跨域问题，结果今天尝试后发现原来配置这么简单，果然还是要真实尝试了才知道。
