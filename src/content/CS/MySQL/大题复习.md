## 题目 3：分组统计（GROUP BY + 聚合函数）

**查询每门课程的平均成绩，显示课程名和平均分，按平均分降序排列。**

```SQL
SELECT c.cname, AVG(sc.score)
FROM sc
JOIN course c ON c.cid = sc.cid
```

## 题目 4：子查询（IN / EXISTS）

**查询选修了“数据库”课程的学生姓名。**

```sql
SELECT s.sname
FROM student s 
WHERE s.sid IN (SELECT sc.sid WHERE sc.cid = 1 FROM sc)
```

```sql
-- 方法1：IN 子查询
SELECT sname
FROM student
WHERE sid IN (
    SELECT sid
    FROM sc
    WHERE cid = (SELECT cid FROM course WHERE cname = '数据库')
);

-- 方法2：多表连接（更简洁，推荐）
SELECT s.sname
FROM student s
INNER JOIN sc ON s.sid = sc.sid
INNER JOIN course c ON sc.cid = c.cid
WHERE c.cname = '数据库';

-- 方法3：EXISTS（效率高，适合大数据）
SELECT sname
FROM student s
WHERE EXISTS (
    SELECT 1
    FROM sc
    WHERE sc.sid = s.sid
    AND cid = (SELECT cid FROM course WHERE cname = '数据库')
);
```
## 题目 5：综合大题（连接 + 分组 + 子查询）

**查询每个学生的姓名、选课门数、最高分，只显示选课门数≥2 的学生，按最高分降序排列。**

```sql
SELECT s.sname, COUNT(sc.cid) AS max_score, MAX(sc.cid)
FROM student s
JOIN sc ON c.sid = sc.sid
GROUP BY s.sid, s.sname
ORDER BY max_score DESC
HAVING COUNT(sc.score) >= 2
```

```sql
SELECT s.sname, COUNT(sc.cid) AS course_count, MAX(sc.score) AS max_score
FROM student s
INNER JOIN sc ON s.sid = sc.sid
GROUP BY s.sid, s.sname
HAVING COUNT(sc.cid) >= 2
ORDER BY max_score DESC;
```

看清楚了，我把 max 的别名写给 count 了

MAX()内部写成 sc.cid 了，应该是 sc.score

JOIN 的 ON 写成 c.sid 了，应该是 s.sid，这种小东西要看清了，我写错好几次了

HAVING 要放在 GROUP  BY 后面，ORDER BY 前面，因为先分组，筛选，全部选完了排序
