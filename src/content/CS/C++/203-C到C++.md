列表初始化
```c++
include <vector>
std::vector<int> a{1,3,5,7};//相当于C里直接初始化列表，这里是初始化了列表a
```

auto
```c++
auto b = 8;//自动变为int
auto c = "hello";//自动变为const char*
//这样短的就自己写了，但变量a这样的类型名字很长，用auto就方便
auto d = a.begin();//a的类型名：std::vector<int>::iterator f，就免得写了
```
 
decltype
```c++
auto varName=value;//auto是根据=右边确定类型
decltype(exp) varName=value;//decltype是根据（）中的变量或者方程或者函数调用返回值来确定varName的类型
//组合使用：
auto fun(int a,int b)->decltype(a / (b + 1.0)){
	return a / (b + 1.0);
}
```

范围 for 循环
```c++
std::string vault ="abcdefg"
for (auto item : valut) {
	print("%c",item)
}//跟Python中 for i in vault 有点像
```
