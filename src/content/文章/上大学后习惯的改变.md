高中的时候周末回家总是抱着个手机刷B站，但是上大学之后看手机的时间多了很多，我反而刷B站刷得少了，倒是文字内容读了很多。先是高中就看的知乎，现在依然看得很多，偶尔看看贴吧，去浙江几个大学的贴吧逛逛，有时候能从互相搬史的帖子之外找到一些有意义的东西，后来我又下载的从前有偏见的小红书，这个就看推送算法了，我从下载开始一直都是搜一些生活相关的事情，不得不说这方面确实很有用，而且时效性强，感觉在线的人很多，用户都比较活跃，刷到的帖子总是有很多新鲜的回复。除此之外，我不知道什么时候开始经常刷微信公众号，我想起因应该是在公众号上刷 github 推荐，一开始刷到韩数同学，后面又关注了逛逛github，然后又给我推了其他内容，刷着刷着推送就符合我的胃口了，直到我刷到了同校的学长开始做的公众号认知增量派，他再 10.30 号发了第一篇公众号，不久前（12.11 号）他决定日更。一开始我是因为他是学长所所以关注的，新发的文章每天都给他点赞，读了一段时间发现他写得是真不错，往前翻看，原来那篇《浙江农林大学马犬事件回应》原创阿树的思考日记认知增量派_2025年11月1日 12:26 浙江_也是他写的，看他的文章看得我心痒痒，想要上手写文章，但我从小不喜欢写作，不是真的不喜欢文字，而是自己不擅长、写不好而不愿意写，不愿意把自己的拙作拿出来献丑。上大学以后我着迷于搜集资源，发现互联网上资源真多啊，我看公众号的分享，逛 github，发现各种开源的好东西，比如 Obsidian, 我从高中开始的写日记的习惯在这里电子化，而且我自己写笔记也在这里写。有一次去 Z-library 的镜像导航站误入了一个资源站，那是真的啥都有，看这这么多资源，我不用都感觉可惜了。还有各种网盘都送初始 10G 的容量，不传什么大东西基本用不完，而且从网上下资源可以保存到网盘下完再把它从网盘删了可以反复用这 10G。还有最近在那个资源站发现的安娜的档案下电子书，然后放 Readest 上读，今天又问 AI 问出来了 github 上有什么学英语的资源，还在那个仓库的 readme 里找到了读英文书、报纸的阅读器。以上种种让我想起了之前在知乎上看到的回答，说的是我们感觉我们这一代黑利吃尽但其实有许多以前的人没有的红利，我此刻对此的感触很深刻，我贴出这篇文章——

> [(21 封私信 / 80 条消息) 你们好陌生人。。你们可以告诉我一个活下去的理由吗？ - 知乎](https://www.zhihu.com/question/546107265/answer/2847869612)

说了那么多，我现在写我这第一篇公众号主要是那个学长的文章的影响，让我看了这么多天却在今天迈出第一步的契机是我又在公众号上刷资源发现了一个 Markdown 排版网站——

> [Markdown | 让排版变 Nice](https://editor.mdnice.com/)


于是在这个不慌不忙的 12.20 的半夜写下了这篇文章，说来我从十一点多写到现在 12.21 号零点了，说来也巧 12.21 恰好是我的生日，今天还恰好是冬至，十九岁生日还是阴历阳历重合的，更巧的是我妈是 11.07 号生日，这一天正好是立冬，我爸是 2.21 号生日，正好是我生日日期的后三位，而且我们三个都属狗，爷爷奶奶一个属龙一个属鸡，都挺巧合的，最巧合的是 19 年前的今天正好诞下了我这颗魔丸。

写到这时间已晚，就让我结束我这篇随意的我的第一篇公众号文章吧！
