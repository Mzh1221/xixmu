## 一、新建虚拟机关键设置（VMware）
1.  新建虚拟机，选择「自定义」，版本选和你的镜像匹配的（比如 `Arch Linux x86_64`）
2.  内存给 `2G` 以上，处理器 `2核` 以上，硬盘选 `60G`（选「将虚拟磁盘存储为单个文件」）
3.  **重点！网络适配器选「NAT 模式」**（共享主机 IP，避免 DNS 坑）
4.  把 Arch ISO 镜像挂载到虚拟机的 CD/DVD 驱动器，勾选「启动时连接」

---

## 二、启动 Live 环境
1.  开机后在 GRUB 界面，直接选默认的 `Arch Linux install medium` 回车
2.  进入 Live 终端，先执行：
    ```bash
    # 开启 VMware Tools，后续可以用右键粘贴
    pacman -Sy open-vm-tools
    systemctl enable --now vmtoolsd
    ```

---

## 三、网络与镜像源配置（先搞定，避免后续坑）
```bash
# 强制设置国内 DNS，彻底解决解析问题
echo "nameserver 114.114.114.114" > /etc/resolv.conf
echo "nameserver 223.5.5.5" >> /etc/resolv.conf

# 强制覆盖为单一国内镜像源
echo "Server = https://mirrors.ustc.edu.cn/archlinux/\$repo/os/\$arch" > /etc/pacman.d/mirrorlist

# 验证网络
ping -c 3 mirrors.ustc.edu.cn
```

---

## 四、分区（`cfdisk` 图形化操作，零难度）
```bash
cfdisk /dev/sda
```
1.  分区表选 `gpt`
2.  新建分区：
    - 第一个分区：大小 `512M`，类型 `EFI System`（`/dev/sda1`）
    - 第二个分区：剩余全部空间，类型 `Linux filesystem`（`/dev/sda2`）
3.  按 `Write` 写入，输入 `yes` 确认，再按 `Quit` 退出

---

## 五、格式化 + 挂载分区
```bash
# 格式化分区
mkfs.fat -F32 /dev/sda1
mkfs.ext4 /dev/sda2

# 挂载分区
mount /dev/sda2 /mnt
mkdir -p /mnt/boot
mount /dev/sda1 /mnt/boot

# 验证挂载
df -h /mnt /mnt/boot
```

---

## 六、安装基础系统
```bash
pacstrap /mnt base linux linux-firmware git
```

---

## 七、生成挂载配置 + 进入目标系统
```bash
genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt /bin/bash
```

---

## 八、目标系统基础配置（可选，先给你列好）
```bash
# 设置时区
ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
hwclock --systohc

# 设置主机名
echo "archlinux" > /etc/hostname

# 设置 root 密码
passwd
```
