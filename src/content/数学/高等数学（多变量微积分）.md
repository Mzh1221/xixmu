# 偏导数
## 抽象复合二元函数求高阶混合偏导（链式法则）
$$
z=f(u,v),u=u(x,y),v=v(x,y)
$$
$$
\text{二阶连续可偏导：}\boldsymbol{f_{12}''=f_{21}''}
$$
简写记法：
$$
\begin{cases}
f_1=\dfrac{\partial f}{\partial u},\quad f_2=\dfrac{\partial f}{\partial v}\\
f_{11}=\dfrac{\partial^2f}{\partial u^2},\ f_{12}=\dfrac{\partial^2f}{\partial u\partial v}\\
f_{21}=\dfrac{\partial^2f}{\partial v\partial u},\ f_{22}=\dfrac{\partial^2f}{\partial v^2}
\end{cases}
$$
$$
\begin{array} { r } { \frac { \partial z } { \partial x } = f _ { 1 } '\cdot \frac { \partial u } { \partial x } + f _ { 2 }' \cdot \frac { \partial v } { \partial x } } \\ { ~ ~ } \\ { \frac { \partial z } { \partial y } = f _ { 1 }' \cdot \frac { \partial u } { \partial y } + f _ { 2 }' \cdot \frac { \partial v } { \partial y } } \end{array}
$$
重点：$\displaystyle f_{1}\text{、}f_{2}$ 仍然是 u、v 的复合函数

所以下一步把一次导的结果的每一项都当成原来的式子用两个 u、v 分别再求导

注意：
- 第二次偏导时后面导是对 y
- 建议一步一步走，把各分部写出来再整合
例题：
$$
z = f ( u , v ) , u = x + y ,v=xy
$$
## 隐函数求偏导
### 一元隐函数
$$
F(x,y)=0
$$
$$
\begin{array} { l l } { { \displaystyle { \frac { d y } { d x } } = - { \frac { F _ { x } } { F _ { y } } } } } & { { \displaystyle { ( F _ { y } \neq 0 ) } } }\end{array}
$$
### 二元隐函数
$$
F(x,y,z)=0,z=z(x,y)
$$

$$
\begin{cases} {\frac{\partial z} {\partial x}=-\frac{F_{x}} {F_{z}}} \\ {\frac{\partial z} {\partial y}=-\frac{F_{y}} {F_{z}}} \\ \end{cases} \left( \boldsymbol F_{z} \neq\mathbf0 \right)
$$
$\displaystyle \text{其中 }F_{x}\text{、}F_{y}\text{、}F_{z}\text{是指原方程全移到一边后对对应的变量求导}$
## 二元函数的连续性和可微性判断
### 逻辑链
$\displaystyle \text{偏导连续}\to\text{可微}\to \text{偏导存在}+\text{函数连续}$
### 定义
$$
\operatorname*{lim}_{( x , y ) \to( x_{0} , y_{0} )} f ( x , y )=f ( x_{0} , y_{0} )
$$
### 判断
1. 代入得到该点函数值 $\displaystyle f(x_{0},y_{0})$
2. 沿任意路径趋近都极限相同才算极限存在
	- 常用路径：$y=kx\text{、}y=kx^2\text{、}x=0\text{、}y=0$
	- 两条路径极限不等 $\Rightarrow$ 极限不存在 $\Rightarrow$ 不连续
## 偏导数存在判定
### 用定义
$$f_{x}(x_{0},y_{0})=\operatorname*{lim}_{\Delta x\rightarrow0}\frac{f(x_{0}+\Delta x,y_{0})-f(x_{0},y_{0})}{\Delta x}$$
$$f_{y}(x_{0},y_{0})=\operatorname*{lim}_{\Delta y\rightarrow0}\frac{f(x_{0},y_{0}+\Delta y)-f(x_{0},y_{0})}{\Delta y}$$
### 直接导
$\displaystyle f_{x}'$ 就对 x 导，y 看成常数导

> [!warning] 分段函数在分界点处必须用定义

$$
\begin{cases}
\dfrac{\partial z}{\partial x}=-\dfrac{F_{x}}{F_{z}} \\
\dfrac{\partial z}{\partial y}=-\dfrac{F_{y}}{F_{z}}
\end{cases}\quad \bigl(\boldsymbol F_{z} \neq \boldsymbol 0 \bigr)
$$

## 多元函数求极值
1. 解方程组的一切实数解，即求出驻点
$$
\begin{cases}
f_x(x,y)=0\\
f_y(x,y)=0
\end{cases}
$$
2. 对每一个驻点 $\left(\,x_{0}\,,\,y_{0}\,\right)$ 求二阶偏导 
$$
\begin{array}{c}{{A=f_{x x}\left(\,x_{0}\,,\,y_{0}\,\right)}}\\ {{B=f_{x y}\left(\,x_{0}\,,\,y_{0}\,\right)}}\\ {{C=f_{y y}\left(\,x_{0}\,,\,y_{0}\,\right)}}\end{array}
$$
3. 看  *$\displaystyle AC-B^{2}$ 如果大于 0 则有极值，此时 A<0 有极大值*，A>0 有极小值，整体小于 0 没有极值，等于 0 不确定

## 积分区域对称的二重积分
如果积分区域 D 关于 x 轴对称，那么如果被积函数关于 y 是奇函数积分就得 0
![[Pasted image 20260606102810.png]]

$$
f{\bigl(}x,-y{\bigr)}=-f{\bigl(}x,y{\bigr)}\Longrightarrow\iint f{\bigl(}x,y{\bigr)}d x d y=0
$$

如果积分区域 D 关于 y 轴对称，那么如果被积函数关于 x 是奇函数积分就得 0

$$
f{\bigl(}-x,y{\bigr)}=-f{\bigl(}x,y{\bigr)}\Longrightarrow\iint{\bigl(}x,y{\bigr)}\,d x d y=0
$$

如果积分区域 D 关于 y=x 对称，称为轮换对称
$$\iint_{D}f(x,y)\,d\sigma=\iint_{D}f(y,x)\,d\sigma$$
> [!warning] 是 x、y 一起交换，不是说 x=y 可以局部随便换

> [!tip] 圆形比较特殊即关于 x 轴对称也关于 y 轴对称，二重积分很容易等于 0

> [!tip] $\displaystyle x^{3},y^{3}$ 等容易看出是奇函数，但xy关于x或者关于y也是奇函数

## 微分方程
### 可分离变量
分离变量把 x,y 拆到等号两边
$$f(x)d x=g(y)d y$$
然后两边积分
> [!warning] 不定积分记得带 C
### 齐次微分方程
类似以下的，各项次数相同
> $$y^{2}+x^{2}{\frac{d y}{d x}}=x y{\frac{d y}{d x}}$$
1. 整理出 $\displaystyle (x+y),xy,\frac{y}{x}$ 这样的复合部分
> $${\frac{d y}{d x}}={\frac{y^{2}}{x y-x^{2}}}={\frac{\left({\frac{y}{x}}\right)^{2}}{{\frac{y}{x}}-1}}$$
2. 令 $\displaystyle \frac{y}{x}=u$，则 $\displaystyle y=ux$，对这个式子导，得到
$${\frac{d y}{d x}}=u+x{\frac{d u}{d x}}$$
3. 取代到方程
> $$u+x{\frac{d u}{d x}}={\frac{u^{2}}{u-1}}$$
4. 让后就可分离变量做
5. 换元记得代回
### 一阶线性微分方程
符合
$$y^{\prime}+P(x)y=Q{\bigl(}x{\bigr)}$$
公式
$$y=e^{-\int P(x) d x}\left(\int Q(x) e^{\int P(x) d x}d x+C\right)$$
如果是齐次的
$$
y^{\prime}+P(x)y=0
$$
则
$$y=C e^{-\int P(x)\,d x}$$
### 常系数齐次方程
含 $\displaystyle y,y',y''\dots,\text{不含}x\text{的题目}$
> $$y^{\prime\prime\prime}-4y^{\prime\prime}+4y^{\prime}=0$$
1. 将 y 变成 r,y 有几 ' r 就是几次
2. 解这个关于 r 的方程（因式分解）
3. 参考表判断解的情况，写出对应的通解

| 特征方程的根                               | 通解                                                               |
| ------------------------------------ | ---------------------------------------------------------------- |
| 单实根 $\displaystyle \alpha$           | $C\cdot e^{\alpha x}$                                            |
| 一对单复根 $\displaystyle \alpha+\beta i$ | $e^{\alpha x}\left(C_{1}{\cos\beta x}+C_{2}{\sin\beta x}\right)$ |
| k 重实根 $\displaystyle \alpha$         | $e^{\alpha x}\left(C_{1}+C_{2}x+\cdots+C_{k}x^{k-1}\right)$      |
实根对照

| 单实根  | $\displaystyle r-1=0\to r=1$                       |
| ---- | -------------------------------------------------- |
| 二重实根 | $\displaystyle (r-1)^{2}=0\to r_{1}=r_{2}=1$       |
| 三重实根 | $\displaystyle (r-1)^{3}=0\to r_{1}=r_{2}=r_{3}=1$ |
|      |                                                    |
4. 以上通解求和得到答案
> [!tips]
> $$
> \begin{align}
> &r_{1}=r_{2}=2,r_{3}=0 \\
> &\text{二重实根：}=e^{2x}\left(C_{1}+C_{2}x\right) \\
> &\text{单实根：}C_{3}e^{0}=C_{3} \\
> &\text{通解：}y=e^{2x}\left(C_{1}+C_{2}x\right)+C_{3}
> \end{align}
> $$
## 二重积分的几何意义
积分区域就是底面，被积函数就是上顶面
> $$\iint_{x^{2}+y^{2}\le R^{2}}\sqrt{R^{2}-x^{2}-y^{2}}\,d\sigma$$ 就是上半球

$$
\begin{align}
f=1\text{地面面积(高度为1)} \\
f(x,y)=\sqrt{R^{2}-x^{2}-y^{2}}\text{半球体积} \\
f(x,y)=\sqrt{x^{2}+y^{2}}\text{圆柱体积}
\end{align}
$$
> [!warning] 对称变成 0 的那个是被积函数为奇对应的那个条件，如果是常数的话就是底乘高
